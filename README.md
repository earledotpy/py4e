# Python for Everyone Exercises

## What
The files contained within are personal Python scripts created by me. They reflect my self-guided learning journey to learn and understand the Python programming language. These files contain Python code I have written from the exercises found in *Python for Everyone* by Charles R. Severance.

## How to Run
Each Python script can be executed within any environment with Python installed. Each file will run a small set of instructions to perform a basic operation (e.g., converting Celsius to Fahrenheit). They contain my working code of the exercise instructions from *Python for Everyone*.

## Why
The files within the repository are the ongoing efforts of my self-directed AI Learning Roadmap. With the help of Anthropic's Claude LLM, I have created a set of learning goals for myself to learn all the different domains needed in order to understand how Artificial Intelligence operates. I am currently on month 6 of the roadmap, which contains an introductory path for beginners to the Python programming language.
